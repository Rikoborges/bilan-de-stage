# 🎓 Bilan de Stage — Ricardo Borges · 6tematik.fr

Présentation interactive du bilan de stage de 2 mois chez **6tematik.fr**.  
Construite en HTML/CSS/JS pur — aucune dépendance, aucune installation.

---

## 📁 Structure des fichiers

```
stage/
├── index.html                ← Page d'accueil (menu de navigation)
├── parte1-wp-php-api.html    ← Présentation Partie 1
├── parte2-woocommerce.html   ← Présentation Partie 2
└── README.md                 ← Ce fichier
```

---

## 🚀 Comment utiliser

1. Télécharger les 3 fichiers `.html` dans le **même dossier**
2. Ouvrir `index.html` dans un navigateur (Chrome recommandé)
3. Appuyer sur `F11` pour le mode plein écran
4. Naviguer avec les boutons ou les touches du clavier

> ⚠️ Les 3 fichiers doivent être dans le même dossier pour que les liens entre eux fonctionnent.

## ✨ Fonctionnalités

- **Navigation** par boutons, dots cliquables et touches du clavier
- **Barre de progression** en haut de chaque présentation
- **Bouton ↑** (retour en haut) sur les slides avec défilement
- **Bouton ← Início** pour revenir à la page d'accueil depuis n'importe où
- **Demos en direct** — appels réels à l'API Chuck Norris (nécessite internet)
- **Syntax highlighting** sur tous les blocs de code PHP et JavaScript
- **Design responsive** — fonctionne sur desktop et tablette
